Le suivi vaccinal, clair, rapide et prêt pour le terrain.

VacciTrack centralise les dossiers patients, les vaccinations, les stocks, les rappels et les certificats PDF dans une interface moderne pensée pour les équipes de santé. Cette page de téléchargement reprend le même langage visuel que l’application : fond profond, cartes vitrées, accents bleus et micro-contrastes nets.

2. Config
Ajoute un `vercel.json` pour rediriger `/` vers cette landing page et garder le site en mode statique.
3. Release
Une fois déployée, remplace les liens de téléchargement par tes assets publics ou un lien GitHub Release.
